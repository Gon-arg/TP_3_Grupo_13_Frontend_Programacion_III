# TP 3 - Programación III Frontend

## Descripcion
Este proyecto es el Trabajo Practico 3 de la materia Programacion III de la UTN. Se trata de una aplicación web frontend que incluye páginas para login, perfil de usuario, servicios, equipo, contacto, FAQ y pedidos.

## Tecnologías Utilizadas
- **HTML5**: Estructura de las páginas web.
- **CSS3**: Estilos y diseño responsivo.
- **JavaScript**: Lógica interactiva y manipulación del DOM.

## Estructura del Proyecto
```
Programacion-III-UTN/
├── README.md
├── index.html                 # Página principal
├── assets/
│   └── img/                   # Imágenes del proyecto
├── css/
│   └── style.css              # Estilos globales
├── js/
│   ├── equipo.js              # Lógica para la página de equipo
│   ├── login.js               # Lógica para el login
│   ├── perfil.js              # Lógica para el perfil de usuario
│   └── servicios.js           # Lógica para servicios
└── pages/
    ├── contacto.html          # Página de contacto
    ├── equipo.html            # Página del equipo
    ├── faq.html               # Preguntas frecuentes
    ├── login.html             # Página de login
    ├── pedido.html            # Página de pedidos
    ├── perfil_usuario.html    # Perfil del usuario
    └── servicios.html         # Página de servicios
```

## Instalación y Ejecución
1. Clona o descarga el repositorio.
2. Abre el archivo `index.html` en tu navegador web preferido.
3. Navega por las diferentes páginas utilizando los enlaces proporcionados.

No se requieren dependencias adicionales ni instalación de paquetes, ya que es un proyecto puro de frontend.

## Funcionalidades
- **Login**: Autenticación de usuarios.
- **Perfil de Usuario**: Gestión del perfil personal.
- **Servicios**: Información sobre servicios ofrecidos.
- **Equipo**: Detalles del equipo de desarrollo.
- **Contacto**: Formulario de contacto.
- **FAQ**: Preguntas frecuentes.
- **Pedidos**: Sistema de pedidos.
